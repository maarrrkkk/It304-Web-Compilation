JS Output

In JavaScript, output refers to the information, results, or content that your program displays for users. There are several ways to control how JavaScript communicates this output:

Common Output Methods:

  1. Console.log:This is the most common and developer-friendly method. It prints the provided data (variables, strings, objects, etc.) to the browser's developer console. This is excellent for debugging and inspecting values during development.

    2. Example Code:

    let name = "Alice";
console.log("Hello", name); // Output: Hello Alice
  2. Alert: This method displays a modal dialog box with the given message, typically used for user interaction or displaying critical information.

    3. Example Code:

    alert("This is an alert message!");
  3. document.write: While not commonly used in modern development due to potential issues with page structure, document.write allows you to directly write content into the current HTML document.

    4. Example Code:

    document.write("This text is written with document.write");
  4. innerHTML:This property targets a specific HTML element and sets its inner HTML content. It's a powerful method for dynamically modifying the content displayed on the webpage.

    5. Example Code:

    let heading = document.getElementById("myHeading");
heading.innerHTML = "New Heading!";

By mastering these output methods, you can effectively control how your JavaScript programs communicate with users and interact with the webpage.
