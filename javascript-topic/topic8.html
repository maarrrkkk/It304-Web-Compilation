JS Functions Parameters

JavaScript function parameters are placeholders for values that a function can accept when it is called. They allow functions to be flexible and reusable by accepting different inputs. Let's discuss the aspects of function parameters:

  1. Syntax
    • Parameters are listed inside the parentheses () of a function declaration or expression.
    • Multiple parameters are separated by commas.
    • Parameters act as local variables within the function's scope.

    2. Example Code:

    // Function declaration with parameters
function greet(name, message) {
    console.log("Hello, " + name + "! " + message);
}
  2. Passing Arguments
    • Arguments are the actual values passed to a function when it is called.
    • The number of arguments should match the number of parameters defined in the function declaration.

    3. Example Code:

    // Calling the greet function with arguments
greet("Alice", "How are you?");
  3. Default Parameters (ES6+)
    • JavaScript supports default parameter values, allowing parameters to have default values if no argument is provided when the function is called.
    • Default parameter values are specified in the function declaration.

    4. Example Code:

    // Function with default parameter
function greet(name = "Guest") {
    console.log("Hello, " + name + "!");
}

// Calling greet with and without argument
greet(); // Output: Hello, Guest!
greet("Bob"); // Output: Hello, Bob!
  4. Rest Parameters (ES6+)
    • Rest parameters allow a function to accept an indefinite number of arguments as an array.
    • Rest parameters are denoted by three dots (...) followed by the parameter name.

    5. Example Code:

    // Function with rest parameter
function sum(...numbers) {
    let total = 0;
    for (let number of numbers) {
        total += number;
    }
    return total;
}

// Calling sum with different number of arguments
console.log(sum(1, 2, 3)); // Output: 6
console.log(sum(5, 10));    // Output: 15
  5. Arguments Object
    • The arguments object is an array-like object available within all functions that contain the values of the arguments passed to the function.
    • It allows functions to accept any number of arguments, even if they are not explicitly defined in the parameter list.

    6. Example Code:

    // Function using the arguments object
function sum() {
    let total = 0;
    for (let i = 0; i < arguments.length; i++) {
        total += arguments[i];
    }
    return total;
}

// Calling sum with different number of arguments
console.log(sum(1, 2, 3)); // Output: 6
console.log(sum(5, 10));    // Output: 15

Understanding JavaScript function parameters enables developers to create more versatile and reusable functions that can handle various inputs and scenarios effectively.
