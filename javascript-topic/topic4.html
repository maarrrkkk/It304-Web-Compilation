JS Statement

In JavaScript, statements are the fundamental building blocks of your programs. They are complete instructions that the JavaScript engine can understand and execute. There are several categories of statements, each serving a specific purpose:

  1. Declaration Statements:These statements introduce variables and functions into your program. They define what kind of data a variable can hold (numbers, strings, etc.) and create reusable blocks of code with functions.

    2. Example Code:

    let age = 30; 
function greet(name) { 
    console.log("Hello", name); 
}
  2. Expression Statements: This method displays a modal dialog box with the given message, typically used for user interaction or displaying critical information.

    3. Example Code:

    let sum = 10 + 20;
console.log(message.toUpperCase());
  3. Conditional Statements: These statements control the flow of execution in your program based on certain conditions. They allow you to make decisions and execute different code blocks depending on whether a condition is true or false.

    4. Example Code:

    if (age >= 18) { console.log("You are eligible to vote"); } 
switch (day) { case "Monday": console.log("Start of the week!"); break; default: console.log("Another day in the week"); }
  4. Looping Statements:These statements allow you to repeat a block of code multiple times. They are essential for iterating over data structures like arrays or performing tasks repeatedly until a certain condition is met.

    5. Example Code:

    for (let i = 0; i < 5; i++) { console.log("Iteration", i); } 
while (userChoice !== "exit") { // ... process user input ... }
  5. Error Handling Statements:These statements help you manage errors that might occur during program execution. They allow you to gracefully handle unexpected situations and prevent your program from crashing.

    6. Example Code:

    try { // Code that might throw an error } catch (error) { console.error("Error:", error.message); }

Remember:

  • Statements are typically terminated with a semicolon (;).
  • Statements can be grouped together within curly braces ({}) to create code blocks.
  • Mastering these different types of statements is crucial for building effective JavaScript programs.
