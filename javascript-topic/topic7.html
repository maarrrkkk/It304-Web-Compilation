JS Functions

JavaScript functions are blocks of code that perform a specific task or calculate a value. They are fundamental building blocks of JavaScript programming, allowing for code reuse, abstraction, and organization. Here's a discussion along with examples:

  1. Function Declaration:Syntax: function functionName(parameters) { // code block }

    2. Example Code:

    // Function to add two numbers
function add(a, b) {
    return a + b;
}
  2. Function Expression : Syntax: var functionName = function(parameters) { // code block }

    3. Example Code:

    // Function expression to subtract two numbers
var subtract = function(a, b) {
    return a - b;
};
  3. Arrow Functions (ES6+):Syntax: var functionName = (parameters) => { // code block }

    4. Example Code:

    // Arrow function to multiply two numbers
var multiply = (a, b) => {
    return a * b;
};
  4. Immediately Invoked Function Expression (IIFE) : Syntax: (function() { // code block })();

    5. Example Code:

    // IIFE to display a message
(function() {
    var message = "Hello, World!";
    console.log(message);
})();
  5. Function Parameters and Arguments: Parameters are placeholders in a function definition and Arguments are the values passed to the function when it's called.

    6. Example Code:

    // Function to calculate the area of a rectangle
function calculateArea(length, width) {
    return length * width;
}
// Calling the function with arguments
var area = calculateArea(5, 3); // length = 5, width = 3
  6. Return Statement : Functions can return values using the return statement.

    7. Example Code:

    // Function to check if a number is even
function isEven(number) {
    return number % 2 === 0;
}
  7. Scope: Functions have their own scope, meaning variables declared inside a function are only accessible within that function (unless using closures).

    Example Code:

    // Global variable
var globalVar = "I'm global";

// Function with local variable
function localFunction() {
    var localVar = "I'm local";
    console.log(globalVar); // Accessible
}

console.log(globalVar); // Accessible
console.log(localVar); // Throws an error

Functions are essential for modularizing code, improving code organization, and promoting reusability. They encapsulate logic, making it easier to understand and maintain codebases, and are a fundamental concept in JavaScript programming.
